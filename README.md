# MongoDB Backend Project 💬

A simple CRUD-based chat application built using Node.js, Express, and MongoDB.  
This project allows users to create, view, edit, and delete chats.

## 🚀 Features
- Create new chats
- View all chats
- Edit existing chats
- Delete chats

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- EJS (Templating Engine)
- CSS

## 📌 Concepts Used
- RESTful Routing
- CRUD Operations
- Method Override (PUT & DELETE)
- Server-side Rendering (EJS)
